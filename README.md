# Initial Email

This little tool just sends an email. Useful for sending a test email out of a cluster.

Can be controlled only via env vars.

Just set the following env vars:

- SMTP_HOST
- SMTP_PORT
- SMTP_USERNAME
- SMTP_PASSWORD
- SMTP_FROM
- SMTP_FROM_NAME
- SMTP_TO
- SMTP_TO_NAME
- SMTP_SUBJECT
- SMTP_TEXT
